# правила использования

### свод правил при использовании расширения UchiX

В UchiX **запрещено**:

- использование нескольких аккаунтов VK для одного аккаунта на учебной платформе.
- решение карточек для других людей (друзей, одноклассников и т. п.) с одного аккаунта VK, за исключением решения карточек родителями детей.
- обратная разработка (reverse engineering) функционала проекта или поиск уязвимостей за пределами программы Bug Bounty.

За нарушение данных правил ваш аккаунт UchiX может оказаться заблокированным автоматической системой.
